# Volt Docs

Documentation for **Volt**, the blazing fast Python stack manager.

## Overview

Volt is an extremely fast, modern template and stack manager for Python projects. Built for speed and simplicity, it automates the boilerplate so you can focus on building.

This repository contains the source code for the official Volt documentation website.

## Quick Start

### Installation

```bash
git clone https://github.com/VishwaKumaran/volt-docs.git
cd volt-docs
bun install
bun run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the project.
