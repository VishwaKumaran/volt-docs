# Contributing

We love contributions! whether it's reporting a bug, suggesting a feature, or writing code.

## Getting Started

1.  Fork the repository: [https://github.com/VishwaKumaran/volt](https://github.com/VishwaKumaran/volt)
2.  Clone your fork.
3.  Install dependencies: `uv sync`

## Pull Requests

Please open a Pull Request against the `main` branch. Provide a clear description of your changes.
