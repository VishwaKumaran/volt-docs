Volt uses "Stacks" to define the structure and dependencies of your generated projects. Currently, Volt focuses on a high-performance **FastAPI** stack, with more coming soon.

## Available Stacks

### [FastAPI](/docs/project-templates/fastapi)
The default stack. A production-ready FastAPI application with modern tooling.
-   **Web Framework**: FastAPI
-   **Server**: Uvicorn
-   **Linting**: Black
-   **Testing**: Pytest

## Upcoming Stacks

-   **Django**: For full-stack applications.
-   **Data Science**: A reproducible environment for notebooks and data pipelines.
-   **CLI Application**: For building command-line tools with Typer or Click.
