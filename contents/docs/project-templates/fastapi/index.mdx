The Volt FastAPI stack is designed to be the best starting point for modern Python APIs.

## Key Features

-   **High Performance**: Built on Starlette and Pydantic.
-   **Type Safe**: Uses standard Python type hints for validation and serialization.
-   **Async First**: Fully asynchronous support for high concurrency.
-   **Auto-Documentation**: Automatic interactive API documentation (Swagger UI / ReDoc).

## Included Tooling

| Tool | Purpose |
| :--- | :--- |
| **FastAPI** | The web framework itself. |
| **Uvicorn** | Lightning-fast ASGI server. |
| **SQLModel** | SQL database interaction (Modern ORM). |
| **Alembic** | Database migrations (via `volt db`). |
| **Docker** | Multi-stage `uv`-based containerization. |
| **Redis** | In-memory data store for caching/pubsub. |
| **TaskIQ** | Distributed background task processing. |
| **Sentry** | Production observability and error tracking. |
| **Pytest** | Robust testing framework. |

## Usage

Select the "FastAPI" option when running `volt fastapi` or initiating a new project.
