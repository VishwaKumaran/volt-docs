# CLI Reference

Volt provides a simple yet powerful CLI to manage your Python projects.

## Commands

-   [`volt fastapi`](/docs/cli-reference/volt-fastapi) - Manage FastAPI projects (create, etc.).
-   [`volt add`](/docs/cli-reference/volt-add) - Add components or features to an existing project.

## Global Options

| Option | Description |
| :--- | :--- |
| `--help` | Show help message and exit. |
| `--version` | Show the version number and exit. |
