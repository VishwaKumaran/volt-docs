# `volt fastapi`

The `volt fastapi` command is the primary entry point for managing FastAPI applications.

## Subcommands

### create

Generate a new FastAPI project with options for production-ready features.

#### Usage

```bash
volt fastapi create [PROJECT_NAME]
```

#### Arguments

-   `[PROJECT_NAME]` (Optional): The name of your project. If not provided, the interactive wizard will prompt you for it.

#### Interactive Options

The creation wizard will prompt you for the following optional features:
-   **Database**: Choose from SQLite, PostgreSQL, or MySQL.
-   **Alembic**: Set up database migrations.
-   **Redis Support**: Include Redis client and lifecycle hooks.
-   **Observability**: Choose from Sentry or Logfire.

#### Example

```bash
volt fastapi create
```
