# `volt add`

The `volt add` command allows you to add features, integrations, or components to an existing Volt project.

## Subcommands

### db

Add database support to the project.

#### Usage
```bash
volt add db
```

Supported databases: SQLite, PostgreSQL, MySQL, and MongoDB and more.

---

### auth

Add authentication boilerplate to the project.

#### Usage
```bash
volt add auth
```

Supported methods: Bearer Token (Authorization Header) or Cookie-based (HTTPOnly).

<Note title="Important">
Authentication requires a database features to be added first. Run `volt add db` if you haven't already.
</Note>

## Example

```bash
cd my-volt-project
volt add auth
```
