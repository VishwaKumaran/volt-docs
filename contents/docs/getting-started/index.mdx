## Volt

**Volt** is a high-performance project generator designed for developers who want clean, reliable, and production-ready Python foundations — instantly. Built for speed and simplicity, it automates all the repetitive setup so you can focus on building real features instead of boilerplate.

Powered by `uv` for instant dependency resolution and environment setup, Volt ensures your projects start on a solid, reproducible foundation.

## Why use Volt?

Volt was built to solve the fragmentation in Python project setup. It unifies package management, linting, formatting, and project structure into a single, cohesive workflow.

With `uv` under the hood, Volt delivers:

-   **Instant Environments** – Setup virtual environments in milliseconds.
-   **Production Ready** – Projects are structured for scalability from day one.

Use it for:

-   **FastAPI Services** – High-performance production APIs.
-   **Data Science** – Reproducible research environments.
-   **Any App Backend** - Fast, scalable backend services.

## Core Features

| Feature                              | Description                                                                 |
| ------------------------------------ | --------------------------------------------------------------------------- |
| **Blazing Fast**                     | Powered by `uv`, install and setup environments 10-100x faster than pip.    |
| **Interactive CLI**                  | Beautiful, easy-to-use prompts to configure your project interactively.     |
| **Modern Stacks**                    | Supports FastAPI (with SQLAlchemy, Pydantic, ...) and more coming soon.     |
| **Best Practices**                   | Pre-configured linting (Ruff), type-checking (MyPy), and testing (Pytest).  |
