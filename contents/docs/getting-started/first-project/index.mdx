title: Creating Your First Project

---

Once Volt is installed, creating a new project is simple. Volt provides an interactive CLI to guide you through the process.

## Command

To create a new FastAPI project, run:

```bash
volt fastapi create my-app
```

This will launch the interactive wizard.

### Step-by-Step Guide

1.  **Database**: Choose your database backend.
    -   **Options**: `None`, `SQLite`, `PostgreSQL`, `MySQL`, `MongoDB`.
    
<Note>
If you select `None`, Volt will generate a basic application structure without database support, and the setup will finish here.
</Note>

2.  **Authentication**: (Only appears if a database is selected)
    -   **Options**:
        -   `None`
        -   `Bearer Token` (Authorization Header)
        -   `Cookie-based` (HTTPOnly)

3. **Alembic**: (Only appears if a database is selected)
    -   **Options**:
        -   `None`
        -   `Yes`

4. **Redis**:
    -   **Options**:
        -   `None`
        -   `Yes`

5. **Observability**:
    -   **Options**:
        -   `None`
        -   `Sentry`
        -   `Logfire`

Volt will then:
-   Create the project directory.
-   Set up the virtual environment.
-   Install all dependencies.

## What's Next?

Navigate into your project folder:

```bash
cd my-app
```

And start the development server:

```bash
uv run uvicorn app.main:app
```

Your API is now running at `http://localhost:8000`.
